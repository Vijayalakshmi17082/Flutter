# Flutter Banking UI Demo

A simple Flutter banking application UI that demonstrates a modern financial dashboard interface. The application shows balance details, debit card information, transaction history, and user profile management.

## Technologies Used

* Flutter
* Dart
* Material UI

## Features

* Balance overview section
* Debit card display
* Transaction history
* User profile page
* Clean and responsive UI design


## Project Structure

```
lib
 ├── main.dart
 ├── screens
 │    ├── home_screen.dart
 │    ├── profile_screen.dart
 ├── widgets
 │    ├── card_widget.dart
 │    ├── transaction_widget.dart
```
